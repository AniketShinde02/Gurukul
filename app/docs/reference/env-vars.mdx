# Environment Variables

The following environment variables are required to run Gurukul. We use these for authentication, real-time media, and performance monitoring.

### Core Infrastructure

- `NEXT_PUBLIC_SUPABASE_URL`: Your Supabase project URL.
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`: Your Supabase anonymous key.
- `SUPABASE_SERVICE_ROLE_KEY`: Service role key for admin operations (Server-side only).

### Media (LiveKit)

- `LIVEKIT_API_KEY`: API Key from LiveKit Dashboard.
- `LIVEKIT_API_SECRET`: API Secret from LiveKit Dashboard.
- `NEXT_PUBLIC_LIVEKIT_URL`: WebSocket URL for LiveKit (e.g., `wss://...`).

### NAT Traversal (TURN)

We use [Metered.ca](https://www.metered.ca) for TURN relaying to ensure 100% video connection rates.
- `NEXT_PUBLIC_TURN_URL`: TURN server URL.
- `NEXT_PUBLIC_TURN_USERNAME`: TURN username.
- `NEXT_PUBLIC_TURN_CREDENTIAL`: TURN password.

### Performance & Safety

- `UPSTASH_REDIS_REST_URL`: URL for Upstash Redis instance (Rate limiting).
- `UPSTASH_REDIS_REST_TOKEN`: Token for Upstash Redis instance.
- `NEXT_PUBLIC_SENTRY_DSN`: Sentry DSN for error tracking.
- `CRON_SECRET`: Random secret used to secure cleanup cron jobs.

### Third-Party APIs

- `NEXT_PUBLIC_GIPHY_API_KEY`: API Key for GIPHY integration in chat.
- `GOOGLE_CLIENT_ID` / `GOOGLE_CLIENT_SECRET`: For Google Social Login.
- `GITHUB_CLIENT_ID` / `GITHUB_CLIENT_SECRET`: For GitHub Social Login.
